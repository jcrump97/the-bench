# AGENTS.md — The Bench

## 1. What This Is

A judicial simulation game where players act as a judge across courtroom cases. Every case is procedurally generated by AI (Gemini). Player decisions (bail, motions, evidence, verdict) affect a reputation score. A lower reputation makes the game harder (more challenges, harsher public reaction).

**Portfolio goal:** demonstrate AI integration into a structured decision system, not just a chatbot. The AI generates content; the player judges it.

## 2. Product Contract (Frozen)

### Session Flow

```
Landing Screen → Generate Case (arrange AI call or demo) → Arraignment
  → Motions → Evidence Admissibility → Verdict → Sentencing → Outcome Summary
  → "Go to Next Case" or "End Session"
```

- **5 cases max per session.**
- **Opt-out after any case.** Player can end session early with "Adjudicate Next Case?"
- **Game over if reputation hits 0** at any point.
- **Session score at end** based on: Legal Compliance, Public Trust, Fairness (simple calc, no AI call needed).
- Offline fallback (demo mode) must produce the same data shapes as Gemini.

### Player Decisions

1. **Arraignment:** set bond type/amount/conditions + reasoning
2. **Motions:** grant/deny/modify defense motions with reasoning
3. **Evidence:** admit/suppress evidence items with reasoning
4. **Verdict:** guilty/not-guilty per charge with reasoning
5. **Sentencing:** pick sentence within range, add conditions

Each decision must:
- Show reputation impact immediately
- Be cached to localStorage before AI responds (mid-case persistence)
- Add a transcript entry describing the ruling

### AI Call Pattern (Mandatory)

Every AI function must follow this shape:

```typescript
async function generateSomething(caseData: SomeInput): Promise<SomeOutput> {
  if (isDemoMode) {
    return deterministicFallback(caseData);
  }
  const result = await callGemini(promptForSomething(caseData));
  return SomeSchema.parse(result); // MUST validate
}
```

No naked Gemini responses. No `as Any`. Parse or throw.

## 9. Resolved Decisions

The following are frozen. Agents implement these directly — no guessing.

### 9.1 Reputation Impact (Scaled by Severity)

Reputation starts at 100. Max 100, min 0. Game over at 0.

| Decision Type | Correct Ruling | Wrong Ruling |
|---------------|---------------|--------------|
| **Arraignment bail** | +3 (low flight risk + bail) to +10 (high flight risk + no bail / high bond) | -5 to -15 (inverse of correct) |
| **Motion ruling** | +3 to +8 (grant meritorious / deny frivolous) | -5 to -10 (wrong side) |
| **Evidence admissibility** | +2 to +5 (admit strong / suppress weak) | -5 to -10 (wrong side) |
| **Verdict** | +5 to +10 (matches facts) | -10 to -15 (contradicts evidence) |
| **Sentencing** | +3 to +5 (within statutory range) | -5 to -10 (outside range) |

**Severity scale:** Low = ±3, Med = ±5-8, High = ±10-15. Use `charge.severity` or `evidence.strength` to determine bucket.

### 9.2 Session Score Formula

Three equal pillars: **33.3% Legal Compliance + 33.3% Public Trust + 33.4% Fairness**

```
Legal Compliance  = (correct_rulings / total_rulings) * 33.3
Public Trust      = (final_reputation / 100) * 33.3
Fairness          = average_case_fairness_score * 33.4
// where case_fairness = 1 - (|actual_sentence - recommended_sentence| / range)

Total Score = round(Legal + Trust + Fairness)
Max score: 100
```

### 9.3 Motion Generation

**Hybrid mode (3C):**
- **Demo mode:** deterministic template pool by charge category. `src/data/demo-motions.ts` maps charge types to motion arrays.
- **Live mode:** AI generates contextual motions via `generateMotions(caseData)` in `src/lib/gemini/service.ts`.
- Both must return identical `Motion[]` shape validated by `MotionSchema`.

### 9.4 Verdict Narrative

**Hybrid mode (4C):**
- **Demo mode:** template string interpolation using case metadata, charges, verdict, and sentence.
- **Live mode:** AI generates contextual narrative via `generateOutcome()` in `src/lib/gemini/service.ts`.
- Both must return `CaseOutcome` shape validated by `CaseOutcomeSchema`.
- Template string example: `"In Case #{docket}, the court found the defendant {guilty/not guilty} on {charge_count} charges. The evidence {was/was not} sufficient to establish guilt beyond a reasonable doubt."`
